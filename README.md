# Utility Angular Library

## Description

**Angular** library for working with websocket (signalR, rxjs, etc.)

## Installation

```bash
yarn add @ngmd/websocket
# or
npm i @ngmd/websocket
```

Library address in [npm](https://www.npmjs.com/package/@ngmd/websocket)
